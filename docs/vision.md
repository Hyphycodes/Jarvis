# Vision

Jarvis is a private AI lifestyle operating system for one user.

It is not a chatbot, not a SaaS dashboard, not a productivity app, not a
notification machine. It is a curator, a planner, a memory, a chief of staff,
a cultural radar, and a long-arc advisor.

## Core surfaces

- **Today** — what matters now
- **Radar** — what is coming into range
- **Circle** — people and relationships
- **North** — long-term direction and identity
- **Index** — hidden searchable archive
- **Voice dock** — always available

## Modes

- **Reactive** — user asks, app plans
- **Anticipatory** — app knows what is coming, app surfaces
- **Generative** — app expands the user's world, app proposes

## The arc

Every meaningful flow should move:
Inspired → Saved → Planned → Prepared → Experienced → Remembered.

## Every surfaced item carries

- context
- reasoning
- a path to action
- a path to memory

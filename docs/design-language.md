# Design language

> The visual system is not finalized. Specific tokens land later, in `/theme`.
> This document captures direction only.

## Palette

- Near-black backgrounds
- Warm ivory typography
- Muted gold accent, used sparingly

## Typography

- Serif-led hierarchy
- Typography-first layouts
- Generous whitespace

## Motion and imagery

- Atmospheric imagery
- Slow, intentional motion
- Minimal icons

## References

- Things 3
- Arc Browser
- Linear
- Readwise Reader
- Cron Calendar

## Anti-patterns

- No bright colors
- No dashboard clutter
- No generic AI UI
- No unnecessary cards

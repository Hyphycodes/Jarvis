# Product principles

## Feel

Editorial. Cinematic. Restrained. Calm. Intelligent. Emotionally aware.
Magazine-like.

## Do

- Surface fewer, better things
- Give every item context, reasoning, an action path, and a memory path
- Move items along the Inspired → Remembered arc
- Treat the user's taste as authoritative
- Respect silence: doing nothing is a valid output

## Don't

- No dashboard clutter
- No bright colors
- No generic AI UI
- No unnecessary cards
- No notification machine behavior
- No SaaS-shaped surfaces
